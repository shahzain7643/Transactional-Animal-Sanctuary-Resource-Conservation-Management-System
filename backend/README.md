# Transactional Animal Sanctuary Resource & Conservation Management System

## Backend Technology
Node.js
Express.js
PostgreSQL
JWT Authentication

## Features
- REST API backend
- JWT authentication
- Role-based access control
- Medical treatment transaction
- Adoption approval transaction
- Inventory management
- Financial ledger system

## API Endpoints

GET /api/v1/animals  
POST /api/v1/auth/login  
POST /api/v1/medical/treat  
POST /api/v1/adoptions/approve  

## Transactions Implemented

### Medical Treatment
- Insert medical record
- Update medicine stock
- Insert inventory transaction

### Adoption Approval
- Update adoption application
- Update animal status
- Insert adoption record
- Insert financial ledger entry
